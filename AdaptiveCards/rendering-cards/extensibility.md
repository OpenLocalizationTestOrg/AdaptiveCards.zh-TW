# <a name="extensibility"></a>擴充性

每個 SDK 都可讓您覆寫任何元素的轉譯, 或甚至加入您定義之全新元素的支援。  例如, 您可以變更`Input.Date`轉譯器以發出您自己的自訂控制項, 同時仍保留轉譯器的其他輸出。 或者, 您可以將自訂`Rating`專案的支援新增至您定義的。

如需範例程式碼, 請展開左側 > 轉譯**卡** -> 上的 [ **sdk** ] 節點, 您要使用  -> 擴充性**的 sdk**
